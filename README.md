# Live Streaming Log Analyzer

A Python & Bash-based tool for analyzing streaming logs, detecting bitrate drops, buffering issues, and encoding errors.

## Features
- **Detects streaming errors:** Scans logs for bitrate drops, buffering issues, and encoding failures.
- **Automated Alerts:** Real-time monitoring with Bash script.
- **User-friendly output:** Displays issues in a structured format using Pandas.
